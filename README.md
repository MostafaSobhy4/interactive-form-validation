# 🚀 Interactive Form Validation

## 📌 Overview
This is an interactive and responsive registration form built using **HTML, CSS, and JavaScript**.  
The project focuses on real-time validation, dynamic error handling, and secure user input management to create a smooth and user-friendly registration experience.

---

## ✨ Features
- Real-time phone input restriction (numbers only)
- Name validation (minimum 3 characters)
- Email validation using Regex
- Phone number validation (minimum 7 digits)
- Country code selector for phone numbers
- Strong password validation:
  - Minimum 8 characters
  - At least one uppercase letter
  - At least one number
  - At least one special character
- Password visibility toggle (show / hide password)
- Dynamic error messages for each input field
- Save registered user data to LocalStorage
- Auto reset form after successful submission
- Clean and responsive UI design

---

## 🛠️ Tech Stack
- HTML5
- CSS3
- JavaScript (ES6)
